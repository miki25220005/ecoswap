# EcoSwap ♻️
EcoSwap is a web platform that rewards users for reducing plastic waste by earning points that can be redeemed for eco-friendly products.

## Features
- Waste submission → EcoPoints
- Leveling system (Bronze → Platinum)
- Product marketplace
- Donation + certificate generation
- Delivery workflow
- PHPMailer password reset
- Admin user management

## Tech Stack
- PHP
- MySQL
- Tailwind CSS
- Composer + PHPMailer

## Folder Structure
